# Experimental Physics Data Analysis

Collection of experimental physics and computational analysis projects involving automation, statistical modeling, signal processing, microscopy, and scientific computing workflows.

## Projects Included

### Quantum Interference & Entanglement Optics Experiment
- Free-space optics alignment and optimization
- Polarization-entangled photon pair generation
- Single-photon coincidence measurements
- Statistical validation of Bell inequality violation
- Single-mode fiber coupling and optical system debugging

### Hall Effect in a Semiconductor
- LabVIEW-Python automation pipeline
- Temperature-dependent semiconductor measurements (80–400 K)
- Non-linear regression and carrier mobility extraction
- Scientific computing and visualization using NumPy/SciPy/Matplotlib
- Automated data acquisition and analysis workflows

### Statistical Signal Analysis & Higgs Boson Modeling
- Log-likelihood maximization and statistical fitting
- Signal and background modeling using polynomial fits
- P-value and Z-score significance analysis
- Confidence interval estimation for Higgs boson mass measurements
- Butterworth filtering and signal sonification workflows
- Numerical optimization using SciPy

### Brownian Motion & Intracellular Transport Analysis
- Automated particle tracking from microscopy data
- Mean-squared displacement (MSD) diffusion analysis
- Stokes-Einstein diffusion modeling
- Kymograph-based intracellular transport analysis
- Statistical mechanics and Langevin dynamics applications
- Experimental uncertainty and systematic error analysis
